[SmartyPants 1.2 is out](http://daringfireball.net/projects/smartypants/).


The big news is that it’s not just for Movable Type anymore. It is now a combination plug-in that you can use, unchanged, for both Movable Type and Blosxom weblogs. It also works as a BBEdit text filter, for educating punctuation in HTML markup that isn’t headed into an MT or Blosxom weblog.


For a full list of changes from version 1.1, see the [change notes](http://daringfireball.net/projects/smartypants/#version_1.2).
