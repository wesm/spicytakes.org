[Erik Barzeski found a very cool feature in Jaguar’s system-wide Script menu](http://nslog.com/archives/2003/02/22/libraryscriptsapplications.php): If you create a sub-folder named “Applications” inside “~/Library/Scripts/”, you can then put application-specific scripts within sub-folders of this folder. These scripts will then appear at the top level of the Scripts menu when that application is frontmost.


In other words, if you put scripts in “~/Library/Scripts/Applications/Safari/”, those scripts will appear at the top level of the menu when Safari is frontmost.


Very useful.
