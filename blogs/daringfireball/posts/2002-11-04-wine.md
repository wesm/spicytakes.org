[Brad Choate makes an interesting argument](http://www.bradchoate.com/past/linuxforwindows.php) — that if the open source [WINE](http://winehq.org) project ever succeeds in its goal to allow Windows applications to run on Linux, that it would ultimately be *bad* for the Linux platform in the long-term:


> I just don’t get it. For running Windows applications, there is a perfectly good OS for doing that: Windows. I’d rather run Linux software on Linux.


Not to mention that the vast majority of Windows software is [craptacular](http://www.julen.net/design/1999/050699.html), and the remainder is [mediocre](http://www.macromedia.com/software/homesite/), at best.
There’s also some [Mac software](http://www.adobe.com/products/adobesupportsXP.html) wearing funny widgets.
