[SmartyPants 1.2.1 is out](http://daringfireball.net/projects/smartypants/). This is a minor update, consisting primarily of small bug fixes for compatibility with Perl 5.005 (version 1.2 used a POSIX-style regex character class, which is only supported in Perl 5.6 or later) and with Blosxom when running in “static” mode.


Coincidentally, Bare Bones Software today [released BBEdit 7.0.3](http://www.barebones.com/company/press.php?news_id=81), which is also primarily a bug fix release. Notable amongst the [bugs fixed](http://www.barebones.com/support/bbedit/current_notes.shtml) are several pertaining to the wrong text encoding being applied to Unix filters and scripts; if you had problems running SmartyPants as a filter in BBEdit 7.0.2, the problems should be fixed by upgrading to BBEdit 7.0.3.
