Yesterday was the first day of [the Lede Program](http://www.journalism.columbia.edu/page/1058-the-lede-program-an-introduction-to-data-practices/906) and so far so awesome. After introducing ourselves – and the 17 students are each amazing – we each fired up an EC2 server on the Amazon cloud (in North Virginia) and cloning a pre-existing disc image, we got an inspiration speech from [Matt Jones](http://www.journalism.columbia.edu/profile/514/10) about technological determinism and the ethical imperative of reproducibility. Then [Adam Parrish](http://www.journalism.columbia.edu/profile/515/10) led the class in a fun “Hello, world!” exercise on the iPython notebook. In other words, we rocked out.


Today we’ll hear from [Soma](http://www.journalism.columbia.edu/profile/517/10) about some bash command line stuff, file systems, and some more basic python. I can’t wait. Our syllabi are [posted on github](https://github.com/ledeprogram/courses).
