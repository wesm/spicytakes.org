Ah! Danny Goodman has released the new version of [Dynamic HTML](http://www.amazon.com/exec/obidos/tg/detail/-/0596003161/joelonsoftware/ref=nosim). It’s been a few years since the first edition, which is *still* the best reference on HTML despite being severely out of date. The new edition is 1400 packed pages that actually tells you what web browsers that are actually in use actually do, which makes it invaluable. It has been brought up to date with all the latest browsers and the newest HTML specs. If you’re working with HTML in any way, shape, or form, this book is an absolute requirement.
