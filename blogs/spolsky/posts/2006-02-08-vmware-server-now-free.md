I have always been a[ huge fan of VMWare Workstation](https://www.joelonsoftware.com/items/2005/05/02.html). “The basic idea is that it lets you run a whole PC inside a window, with its own operating system and hard drive.”


I’ve [dreamed](https://www.joelonsoftware.com/items/2003/06/02.html) of the day when I could use something similar for servers. VMWare has had a couple of server products for a while now, GSX and ESX, but they were expensive and complicated.


No longer. [VMWare Server is going free](http://www.vmware.com/products/server/). Wow. Mike Gunderloy [explains](http://www.larkware.com/editorials/vmware.html) why this is “a market-changing move.” This is great stuff, and it’s not half baked.
