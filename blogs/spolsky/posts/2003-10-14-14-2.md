**PHP and Unicode**


Scott Reynen [shows](http://www.randomchaos.com/document.php?source=php_and_unicode) how to use integer arrays in PHP to handle Unicode manually.


There are also functions available called the Multi-Byte String Functions which handle [many encodings](http://www.php.net/manual/en/ref.mbstring.php). Please ignore the part in the documentation that says that this is “developed to handle Japanese characters.” It actually appears to handle lots of encodings including the Unicode ones. However it is [turned off by default](http://www.zend.com/lists/php-dev/200211/msg00210.html) so you must recompile PHP to enable it.
