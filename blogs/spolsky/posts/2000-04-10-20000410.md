Most of the hard core C++ programmers I know hate user interface programming. This surprises me, because I find UI programming to be quintessentially easy, straightforward, and fun.  Now online:  Chapter One of [User Interface Design for Programmers](https://www.joelonsoftware.com/uibook/chapters/fog0000000057.html), a gentle introduction with the fundamental axiom of good user interface design.  [Chapter Two](https://www.joelonsoftware.com/uibook/chapters/fog0000000058.html) shows how to apply this axiom and gives some real life examples.  In coming weeks there will be at least six more chapters.
