**And You Thought You Had Source Control Problems** Ray Ozzie sent me a link to [a presentation given by Mark Lucovsky](http://www.usenix.org/events/usenix-win2000/invitedtalks/lucovsky_html/), a developer at Microsoft, about the Windows NT development process.  29 million lines of code. Source code takes up 50 gigabytes. Build takes 8 hours on a 4 CPU machine. 1000 machines in the “stress test” farm. No, they don’t use SourceSafe.  But I don’t think CVS could handle it, either.
