What would the world be like **if Windows 95 and Windows NT has been cancelled?** You’d be surfing the web right now in [Windows for Workgroups 3.11](http://en.wikipedia.org/wiki/Windows_3.x):


![](https://blog.codinghorror.com/content/images/2025/03/image-293.png)


I absolutely love the fact that **we get a javascript error on the Google homepage**. Not unreasonable, considering WFW 3.11 was well on its way to total obsolescence before Google even made it out of a Stanford dorm room [in 1998](https://blog.codinghorror.com/google-hardware-circa-1999/).


I never browsed the intarweb through any version of Windows 3.x. Most people are surprised to find that there was a Windows 3.x version of circa-1999 Internet Explorer 5. By the time the web caught on in late ’95, most of the early adopters I knew had already transitioned to Windows 95. I distinctly remember using the first version of IE under Windows 95 – the one that didn’t understand the <table> tag or animated gifs!


As for the title of this post, according to the canonical [list of Microsoft code names,](https://web.archive.org/web/20051124162805/http://bink.nu/Codenames.bink)** Snowball** was the internal Microsoft codename for the WFW 3.11 project. If you want to rock your own WFW 3.11 virtual machine, you can [download the OS from peteweb](https://web.archive.org/web/20051016224722/http://www.peteweb.com/index.php?showtopic=3987). And don’t forget DOS 6.22 like I did! Set it up following the great instructions from Virtual PC guy:

- [Installing Windows for Workgroups 3.11](https://web.archive.org/web/20051026085853/http://blogs.msdn.com/virtual_pc_guy/archive/2004/10/26/247793.aspx)
- [Networking Windows 3.11](https://web.archive.org/web/20061029061817/http://blogs.msdn.com/virtual_pc_guy/archive/2004/10/28/248940.aspx)


There’s a huge collection of old 16-bit Windows software on the [10 year anniversary](https://web.archive.org/web/20070701091708/http://www.freewebs.com/win3x/download.htm) of Windows 3.1 page. It’s a sketchy looking website, but if you follow the yahoo briefcase download instructions closely, the software is all still present and downloadable.


You may find [WinImage](http://www.winimage.com/winimage.htm) helpful here for creating floppy disk images from collections of files. Virtual PC only understands non-compressed (.IMA) floppy images, so be sure you avoid the default compressed (.IMZ) floppy images.
