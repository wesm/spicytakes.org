I’m a huge fan of [Beagle Brothers](https://blog.codinghorror.com/our-programs-are-fun-to-use/) style cheat sheets, because nothing promotes the illusion of mastery like a **densely packed chart of obscure reference information**:

|  |  |
|  |  |


kg-card-end: html

Just throw some of those babies up on your walls and people will know that they’re clearly dealing with a *coding genius!*


VisiBone makes great modern equivalents of these classic references, and they’re available in dense card, dense foldout, large print book, and huge wall chart formats. The Visibone regular expression reference is the best concise regex reference I’ve found to date – and there’s a [free online version](http://www.visibone.com/regular-expressions/), too. It is limited to the JavaScript regex syntax, but that’s 98 percent of what most people will need.


I’ve seen the CSS Cheat Sheet before, but I didn’t realize there’s an **entire set of cheatsheets freely available from **[**the same site**](http://addedbytes.com/cheat-sheets/):

- [ASP/VBscript](http://addedbytes.com/cheat-sheets/asp-vbscript-cheat-sheet/)
- [HTML Character Entities](https://web.archive.org/web/20081204152615/http://addedbytes.com/design/html-character-entities-cheat-sheet/)
- [JavaScript](https://web.archive.org/web/20081120161310/http://addedbytes.com/javascript/javascript-cheat-sheet/)
- [MySQL](https://web.archive.org/web/20081204135936/http://addedbytes.com/php/mysql-cheat-sheet/)
- [Apache mod_rewrite](http://addedbytes.com/cheat-sheets/mod_rewrite-cheat-sheet/)
- [CSS](https://web.archive.org/web/20081204152056/http://addedbytes.com/css/css-cheat-sheet/)
- [PHP](https://web.archive.org/web/20081204134600/http://addedbytes.com/php/php-cheat-sheet/)
- [RGB Hex Color](https://web.archive.org/web/20081204113036/http://addedbytes.com/resources/colour-chart/)


These are all conveniently provided in PDF or PNG formats. If you’re a Microsoft.NET programmer, there’s no shortage of cheatsheets for you as well:

- [Visual Basic 2005 Keyboard Reference](http://www.microsoft.com/downloads/details.aspx?FamilyID=6bb41456-9378-4746-b502-b4c5f7182203&DisplayLang=en)
- [Visual C# 2005 Keyboard Reference](http://www.microsoft.com/downloads/details.aspx?FamilyID=c15d210d-a926-46a8-a586-31f8a2e576fe&DisplayLang=en)
- [Visual C++ Keyboard Reference](https://web.archive.org/web/20070220171308/http://www.microsoft.com/downloads/details.aspx?FamilyID=bccf84f4-4136-48b2-b4ec-83eaa484da20&DisplayLang=en#filelist)
- [Format String](https://web.archive.org/web/20070203065801/http://john-sheehan.com/blog/wp-content/uploads/msnet-formatting-strings.pdf)
- [ASP.NET 2.0 Page Lifecycle and Events](https://web.archive.org/web/20070206015921/http://john-sheehan.com/blog/wp-content/uploads/aspnet-life-cycles-events.pdf)
- [Visual Studio 2005 C# code snippets](https://web.archive.org/web/20070127172504/http://john-sheehan.com/blog/wp-content/uploads/vs2005-code-snippets.pdf)


If you don’t see the one you want, here are few other sites that aggregate cheatsheets. Many of them are open-source / UNIX oriented, but they run the gamut.

- [Dr. Web’s Cheat Sheet List](https://web.archive.org/web/20060602082510/http://www.drweb.de/weblog/weblog/?p=571)
- [Bluewin’s Cheatsheet Sites](https://web.archive.org/web/20060117025343/http://mypage.bluewin.ch/yuppi/links/cheatsheets.html)
